# The Java programming language has around 30 operators as summarized below:
```
Simple assignment
=

Arithmetic
+   -   *     /   %

Unary
+   -   ++     -- !

Relational
==   !=   >     >=   <   <=

Conditional
&&     ||   ? : (ternary)

Type comparison
instanceof

Bitwise and Bit shift
~     <<   >>   >>>   &     ^   |
```
